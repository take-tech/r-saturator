# R-Saturator

R-Saturator is a lightweight JUCE/CMake audio plugin for gentle saturation.
The first prototype focuses on a clean signal path, five practical controls,
and DAW-loadable VST3/AU builds.

## Features

- Input Gain, Drive, Tone, Output Gain, and Mix parameters
- Lightweight soft clipping saturation
- Simple Tone control for darker/brighter response
- Dry/Wet Mix for blending the processed signal with the original
- APVTS-backed parameters for host automation
- Minimal five-knob JUCE editor

## Supported Formats

- VST3
- AU on macOS

## Build

JUCE must be available at `external/juce`.

```sh
cmake -S . -B build -DCMAKE_BUILD_TYPE=Release
cmake --build build --config Release
```

The VST3 target is generated by CMake/JUCE. On macOS, the AU target is also
generated.

## Development Status

Current version: v0.1.0 prototype.

This version is intentionally small: no oversampling, preset browser, custom
LookAndFeel, or advanced modulation yet. The goal is a stable first plugin that
loads in a DAW and provides light saturation with the core five parameters.
